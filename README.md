# Tic-Tac-Toe Game (React)

A web-based Tic-Tac-Toe game built with React to demonstrate component-based design, state management, and reusable UI elements.

## Technologies
- React
- JavaScript
- HTML / CSS

## Features
- Two-player game
- Interactive UI with state updates
- Responsive layout

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to launch locally
